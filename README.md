# React SPA width gallery component
Personal sculptor's site 

### Under the hood
  - React
  - Semantic UI React
  - react-masonry-component
  - React Router 
 - ES6
 - No DOM manipulation libraries used (such as jQuery)
 
 ### Installation
Install the dependencies and devDependencies and start the server

```sh
$ git clone
$ npm install 
$ npm start
```
